# randomDataGenerator

Simple browser app for generating random identifiers.

Features:
- Configurable prefix, suffix, middle length, and number of generated items
- Generate on click
- Copy all generated values with one click
- Responsive UI
- Batched generation for better responsiveness on large counts

How to use:
1. Open [index.html](index.html) in a browser.
2. Set prefix, middle length, count, and suffix.
3. Click Generate.
4. Click Copy All if you want the full output in your clipboard.

Notes:
- The middle part currently uses digits 0-9.
- Duplicate probability is an estimate based on the selected length and count.
